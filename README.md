# 空想ラジオ局 | Imaginary Radio Station

眠れない夜に寄り添う、架空の周波数。日常の喧騒から少し離れて、静かな夜の海に潜るような時間をお届けするYouTubeチャンネルの公式ポータルサイトです。

## 制作環境 // PRODUCTION ENVIRONMENT

本プロジェクトでは、人間のクリエイティビティとAI技術の融合を試みています。企画・世界観・脚本などの核となる部分は人間が手作業で行い、AIを強力な補助ツールとして活用しています。

### 使用ツール一覧

#### VIDEO
- **DaVinci Resolve**
  - 映像編集、カラーグレーディング、合成処理に使用。

#### IMAGE
- **Affinity Photo / Designer**
  - 画像編集、ロゴデザイン、最終的なビジュアル調整に使用。
- **Nano Banana (Flow)** [★AI]
  - コンセプトアート、キャラクターのベース画像生成に使用。
- **ChatGPT (DALL-E 3)** [★AI]
  - アイデアの具体化、補助的な画像生成に使用。

#### AUDIO
- **Suno** [★AI]
  - 各ステーションのBGM制作、架空の楽曲生成に使用。
- **Google AI Studio** [★AI]
  - TTS（音声合成）を用いたDJのナレーション生成に使用。
- **Voicemod**
  - 生成した音声の加工、エフェクト処理に使用。

#### WEB
- **Antigravity**
  - プログラミング、Webサイトの構築・デバッグに使用。
- **Claude** [★AI]
  - コードの最適化、フロントエンド実装の補助に使用。

## 技術スタック

- **Design Concepts**: Cyberpunk, Retro-futurism, CRT Effects
